📡 Log Streaming Pipeline
Both engines push build logs to Redis Pub/Sub during build. API server subscribes and emits to frontend via SSE.
Worker → Redis Pub/Sub → API Socket Sub → SSE Stream → Frontend EventSource
channel: logs:{deploymentId} X-Accel-Buffering: no 500ms poll fallback
🔐 Env Variables
AES-256-GCM encrypted at rest in Postgres. Decrypted in-memory only at build time. Injected as env into Docker container.
AES-256-GCM per-project key --env-file injection
🔁 GitHub Webhook
Auto-deploy on git push. POST /webhooks/github → validate HMAC-SHA256 → enqueue new build job → SSE notifies frontend.
HMAC-SHA256 auto-redeploy X-Hub-Signature-256
